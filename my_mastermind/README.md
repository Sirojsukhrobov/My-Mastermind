# Welcome to My Mastermind
***

## Task
Here I should create a game that is named "mastermind". In this game coder gives code that contain four different numbers. 

## Description
Mastermind is a game composed of 8 pieces of different colors.
A secret code is then composed of 4 distinct pieces.

The player has 10 attempts to find the secret code.
After each input, the game indicates to the player the number of well placed pieces and the number of misplaced pieces.

Pieces will be '0' '1' '2' '3' '4' '5' '6' '7'.

If the player finds the code, he wins, and the game stops.
A misplaced piece is a piece that is present in the secret code butthat is not in a good position.

You must read the player's input from the standard input.

## Installation
gcc -o my_mastermind my_mastermind.c

## Usage
It get 4 numbers which is accurate color, and it compares with input number
Well placed pieces: 0
Misplaced pieces: 4
